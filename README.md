### Run the App Locally

1. Clone the repository:

2. Navigate to the project directory:

```bash
cd python-chat
```

3. Run init script:


```bash
make init
```

4. Activate the virtual environment:

```bash
source .venv/bin/activate
```
5. Up the database (using docker-compose):
```
docker compose up -d
```

7. Run the app:

```bash
python run.py
```
